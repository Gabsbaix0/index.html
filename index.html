Manipulando Arrays em JavaScript

Guia prático com exemplos interativos

shift()

Remove o primeiro item do array.

A primeira pessoa da fila foi embora!

Exemplo 1

let frutas = ["maçã", "banana", "laranja"];
frutas.shift(); 

            

          

        


        

          

            
Exemplo 2

            
let numeros = [10, 20, 30];
numeros.shift();

          

          

            
            

              

            

          

        


        

          

            
Exemplo 3

            
let letras = ["x", "y", "z"];
letras.shift();

          

          

            
            

              

            

          

        

      

    


    
    

      
unshift()

      
Adiciona um ou mais itens no início do array.

      
Alguém furou a fila!

      
      

        

          

            
Exemplo 1

            
let frutas = ["banana", "laranja"];
frutas.unshift("maçã");

          

          

            
            

              

            

          

        


        

          

            
Exemplo 2

            
let numeros = [2, 3];
numeros.unshift(1);

          

          

            
            

              

            

          

        


        

          

            
Exemplo 3

            
let letras = ["b", "c"];
letras.unshift("a");

          

          

            
            

              

            

          

        

      

    


    
    

      
concat()

      
Junta dois ou mais arrays em um novo array.

      
Duas filas se unindo.

      
      

        

          

            
Exemplo 1

            
let a = [1, 2];
let b = [3, 4];
let c = a.concat(b);

          

          

            
            

              

            

          

        


        

          

            
Exemplo 2

            
let x = ["x"];
let y = ["y", "z"];
let resultado = x.concat(y);

          

          

            
            

              

            

          

        


        

          

            
Exemplo 3

            
let nomes1 = ["Ana"];
let nomes2 = ["Bia", "Carlos"];
let todos = nomes1.concat(nomes2);

          

          

            
            

              

            

          

        

      

    


    
    

      
flat()

      
Achata arrays aninhados (arrays dentro de arrays).

      
Tirar caixas de dentro de caixas.

      
      

        

          

            
Exemplo 1

            
let arr1 = [1, [2, 3]];
arr1.flat();

          

          

            
            

              

            

          

        


        

          

            
Exemplo 2

            
let arr2 = [1, [2, [3, 4]]];
arr2.flat(2);

          

          

            
            

              

            

          

        


        

          

            
Exemplo 3

            
let arr3 = [["a", "b"], "c"];
arr3.flat();

          

          

            
            

              

            

          

        

      

    


    
    

      
splice()

      
Remove, substitui ou adiciona elementos no array original.

      
Abrir espaço no meio da fila pra adicionar ou remover pessoas.

      
      

        

          

            
Exemplo 1

            
let nomes = ["Ana", "João", "Carlos"];
nomes.splice(1, 1, "Maria");

          

          

            
            

              

            

          

        


        

          

            
Exemplo 2

            
let numeros = [1, 2, 3];
numeros.splice(1, 0, 1.5);

          

          

            
            

              

            

          

        


        

          

            
Exemplo 3

            
let letras = ["a", "b", "c"];
letras.splice(0, 2);

          

          

            
            

              

            

          

        

      

    


    
    

      
slice()

      
Copia parte do array sem modificar o original.

      
Cortar uma fatia do bolo sem mexer no resto.

      
      

        

          

            
Exemplo 1

            
let letras = ["a", "b", "c", "d", "e"];
letras.slice(1, 4);

          

          

            
            

              

            

          

        


        

          

            
Exemplo 2

            
let numeros = [10, 20, 30, 40];
numeros.slice(2);

          

          

            
            

              

            

          

        


        

          

            
Exemplo 3

            
let frutas = ["maçã", "banana", "laranja"];
frutas.slice(0, 2);
